## 📋 What is MakuTweaker?

MakuTweaker is a simple application designed for Windows users who want to adjust system settings easily. It helps improve your computer’s performance and customize Windows behavior without needing technical skills or programming knowledge. This software lets you change common tweaks quickly and safely.

This repository holds the source code. You do not need this code to run the app unless you want to build it yourself. For most users, downloading the ready-to-use version is best.

## 🖥️ System Requirements

MakuTweaker works on Windows 10 or later. Make sure your PC meets these requirements:

- Windows 10, Windows 11, or later  
- 4 GB RAM or more  
- 100 MB free disk space

The app runs on both 32-bit and 64-bit versions of Windows.

## 🎛 Using MakuTweaker: Step-by-Step

1. Choose a category like Performance, Startup, or Privacy.  
2. Review the available tweaks. Each one has a short description.  
3. Select the tweak you want by clicking its checkbox or toggle.  
4. Click **Apply Changes** at the bottom of the window.  
5. Some tweaks may require a restart to take effect. The app will notify you if this is needed.  
6. To undo changes, use the **Restore Backup** option from the menu.
